# QReel

Tool to download reels from instagram by URL writen in go

## Usage as a CLI

### Installation
```shell
go install github.com/qerdcv/qreel
```

### Usage
```shell
qreel -url https://www.instagram.com/reel/CvpqeLsJB4J/?igshid=MzRlODBiNWFlZA== -o output.mp4
```
